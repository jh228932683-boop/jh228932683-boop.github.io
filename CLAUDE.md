# 个人主页项目

一个简洁极简风格的中文个人主页，风格参考苹果官网：
大量留白、字体优雅、动画克制、内容聚焦。

## 技术栈
- HTML + CSS + JavaScript
- 纯静态页面，无需后端
- 所有代码写在一个 index.html 文件里

## 页面结构（按顺序）
1. 顶部导航：姓名/logo 在左，导航链接在右
2. Hero 区域：一句话介绍自己，简短有力
3. 关于我：2-3 段文字，介绍背景和兴趣
4. 项目展示：卡片式布局，每张卡片包含项目名、描述、标签
5. 联系方式：邮箱、微信、社交媒体等
6. 页脚：简单一行版权信息

## 设计规范
- 风格：极简，大量留白，苹果官网风格
- 背景色：纯白 #FFFFFF 或米白 #FAFAFA
- 主色调：黑色文字 #1d1d1f
- 强调色：一种颜色即可，不超过两种
- 字体：系统默认字体（-apple-system, PingFang SC）
- 动画：只用轻微的 fade-in 和 hover 效果，不花哨
- 语言：全部中文

## 响应式要求
- 需要适配手机和电脑两种屏幕
- 手机端导航折叠成汉堡菜单

## 注意事项
- 所有代码写在 index.html 一个文件里（CSS 和 JS 都内嵌）
- 注释用中文
- 项目展示区域的内容先用占位假数据填充，我之后会告诉你真实内容
- 联系方式先留空，等我填写
- 不要用任何外部 UI 框架（不用 Bootstrap、Tailwind）
- 可以用 Google Fonts 或系统字体

## 运行方式
直接用浏览器打开 index.html 即可预览

## 错误日志

### 2026-05-09：关于我区块居中修复不完整
- **问题：** 修复 `#about` 区块居中时，只添加了 `justify-content: center` 和 `max-width`，
  但忘记了删除 `margin-left: 32px; margin-right: 32px`，导致提交不完整。
  后续恢复文件时该问题暴露出来，需要重新修复。
- **教训：** 修改 CSS 布局属性时，必须全面检查元素的所有定位相关属性
  （margin、padding、position、display 等），确保新旧样式规则没有冲突或遗漏。
  涉及定位的修改完成后，应当在浏览器中实际预览确认效果再提交。
